# Cypress_Swag-Labs
# Prerequisite
Node.js

Visual studio code

Good internet connection

How to set up this project
Install Node.js

Install Visual Studio code

Create folder for cypres project 4.open the folder in vs code

open vs code terminal and run the command:

```bash
npm init -y
```

Install cypress

npm install cypress

```bash
npx cypress -v
open cypress
```
```bash
npx cypress open
```
